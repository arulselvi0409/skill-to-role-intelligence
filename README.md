# Skill-to-Role Intelligence Platform

A web application that analyzes user skills, maps them to relevant job roles,
shows skill gaps, estimates salary growth, and generates a downloadable PDF report.

## Features
- Skill-to-role matching with percentage score
- Improvement suggestions and project ideas
- Salary comparison (current vs after upskilling)
- Downloadable PDF report

## Tech Stack
- HTML
- CSS
- JavaScript
- jsPDF
